    Procedures

    1. Check data (esp. last one) and acid usage
    2. Check flow rate (>80 mL/min) on GasBench (2) to decide measure/re-flush the next batch
    3. Check acid pump (esp. Monday)
      1. Forward
      2. manually click button on acid pump multiple times: should be <4s?? per drop
      3. wipe off acid
      4. consider changing Acid Pump–Calibrate...
    4. Put on dummies/standards/samples
    5. Needle maintenance
      1. magnifier
        • curve is original/smooth
        • hole is clear
      2. use little drill to clean: poke in <1 mm, no need to be too invasive unless last batch:
        • first two peaks are smaller than latter peaks
        • peaks do not gradually decrease one by one
      3. put capillaries behind the arm of auto-sampler and put back needles
    6. Aquisation: choose and edit .seq file
      • usually use Carbonate Acid Measure 50 Extra8.seq (with Acid Drop & Measure.met), if more acid drops are needed for some samples with less TIC, use Carbonate Acid Measure 50 Extra8 diff drops.seq and change method into, for instance, Acid Drop & Measure 12 drops.met for these samples
      • input Identifier 1 for samples, Identifier 2 (weight) for standards
      • Options: update date in the Folder Name and Export File Name
      • Save
    7. Check: Ref CO2 On/Off (esp. Monday)
    8. Start
    9. Check first bypass: peaks should gradually decrease one by one

    Reboot auto-sampler

    • disconnect power cord, reconnect

      • remove needles

    Replace He (ca. 6 months)

    1. turn off source, needle valve
    2. old cylinder: turn off valve, disconnect, write: “MT”
    3. new cylinder: connect (using a short spanner to prevent over-tightness), 3 bar (>1 bar for regulator in GasBench), leak check, write date
    4. wait 15 mins, turn on needle valve
    5. turn on source

    Replace filament (#1027920)
