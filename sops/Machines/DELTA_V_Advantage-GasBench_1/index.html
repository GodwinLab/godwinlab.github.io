    DELTA V Advantage–GasBench (1)

    Sample preparation

    • Sample vial: disposable 12 ml borosilicate glass vial (Labco 9RK8)
    • Sample size: 100–200 µg
    • Sequence (16 standards + 50 samples), 96 positions:
      • 8 bypass (last run Carrara Z)
      • Carrara Z × 3, Fletton clay
      • 8 bypass (6 last run Carrara Z, 2 last run Fletton clay)
      • (10 samples, Carrara Z × 2) × 4, 10 samples, Fletton clay, Carrara Z × 3
      • 8 dummies for acid drop
      • 6 acid reservoirs

    Procedures

    1. Check data (esp. last one) and acid usage
    2. Check acid pump (esp. Monday)
      1. Forward
      2. manually click button on acid pump multiple times: should be <4s?? per drop
      3. wipe off acid
      4. consider changing Acid Pump–Calibrate...
    3. Put on dummies/standards/samples
    4. Needle maintenance
      • magnifier
        • curve is original/smooth
        • hole is clear
      • if last batch has abnormal peak shape/location??, use little drill to clean (~1 mm)
      • put capillaries behind the arm of auto-sampler and put back needles
    5. Aquisation: choose and edit .seq file
      • usually: extra8
      • input Identifier 1 for samples, Weight for standards
      • use a different method (e.g.: Acid Drop & Measure 6 drop ) with more acid drops for samples with less TIC
      • Options: update date in the folder’s name
      • Save
    6. Check: Ref CO2 On/Off (esp. Monday)
    7. Start
    8. Check first bypass: peaks should gradually decrease one by one

    Flush on GasBench (2)

    1. Needle maintenance…
      • …
      • check flow rate
        • if < 80 and/or last batch has abnormal peak shape/location??, use little drill to clean (~1 mm)
      • … put back needle and press OK
    2. Sequence: GAS FLUSHFILL MISC SINGLE NEEDLE
    3. First one:
      1. Check out flow from the glass capillary
      2. Check on/off results of the working gas (routine for DIC analysis)

    Misc.

    • Reboot auto-sampler:
      • disconnect power cord, reconnect
      • Check positions
        • remove needles
        • Menu > Utilities > F3 > Object Trays > Tray 01
    • Change fan
      • push and pull out/insert the cable
    • Nafion for water removal
    • Replace He (ca. 6 months)
      1. turn off source, needle valve
      2. old cylinder: turn off valve, disconnect, write MT
      3. new cylinder: connect (using a short spanner to prevent over-tightness), 3 bar (>1 bar for regulator in GasBench), leak check, write date
      4. wait 5 mins, turn on needle valve
      5. turn on source
    • Replace filament (#1027920)
      1. turn off: source, needle valve, pumps (on the panel, hear the turbo pump down)
      2. open door by removing 2 screws
      3. unplug 3 cables without damaging the PCB
      4. loose 4 screws
      5. remove 3 screws
      6. press source when loosening the 4th screw to prevent it popping out
      7. replace filament
        1. gloves on
        2. take off 4 screws
        3. half tighten 2 screws, push filament up left, fully tighten 2 screws
        4. insert two leads (+/−) into the slots
        5. press and check the metal connections
        6. check multimeter
      8. monitor high vacuum ( Instrument control)
        • Scan Time: 100000?, Delay: 0.1
        • 1e-6 → <1e-7
      9. turn on pumps (on the panel)
      10. turn on source (needle valve off): high vacuum increases, and then return back
      11. turn off source
      12. turn on needle valve high vacuum ≈1e-6
      13. note down previous sensitivity
      14. Auto focus
      15. save
      16. Pass to gasconfiguration
      17. run CO2 OnOff Sept2014.seq (100 measurements, save in ... CO2 ONOFF)
        • BGD44: 14 → 2.4 mV
        • sensitivity: 4500 → 4100
        • check linearity between different cycles within one measurement (1–10V, step?, ??<0.03‰/V)
