    Sample preparation

    • Sample vial: reusable 1 ml borosilicate glass vial (WHEATON W986284NG) + black cap + blue septa (Analytical Columns C234115)
    • Sample size: 100–150 µg
    • Sequence (standards 12 + samples 30): Carrara Z × 3, Atlantis II, 10 samples, Carrara Z × 2, 10 samples, Carrara Z × 2, 10 samples, Atlantis II, Carrara Z × 3

    Procedures

    Monday

    1. Prep–Water Trap Chiller On
    2. AC, ML, MH
    3. Test acid drop
      1. Put tissue under needle
      2. Prep–Dispense Acid
      3. Check acid drop after ~5 mins. If not, repeat the last step.
      4. Redo 2 & 3
    4. 45–60 mins

    Monday to Thursday

    1. Check & prepare
      1. Log on the screen: Finished Autorun
      2. Log paper
      3. Short press LF/FF button several times to feed paper forward until 4–5 holes are out
      4. Remove LN2 tubes from dewar
      5. Check 2 LN2 tube connections to cold fingers
      6. Put lids on samples
    2. Flush sample needle with He gas
      1. Prep–Move Gilson To Next Vial. If Gilson does not move: Prep–Initialise Gilson, then try again.
      2. AB, MH
      3. Check: AD
      4. AE
      5. Connect He gas, open cylinder valve, open regulator valve (<2 bar)
      6. Wipe off acid
      7. Check flow rate through the needle
      8. Disonnect He gas
    3. Pump out sample side
      1. Put on an old vial on needle: holes inside the vial & DO NOT touch the acid
      2. ML, AD, until ML vac ≈ 1.0e-3 (take out old sample vials in the heater block)
      3. Prep–Initialise Gilson
      4. Remove old vial
      5. AB, ML
      6. MH (auto ML)
    4. Refill reference gas
      1. Set reference gas bellow to 3100 (Stage 2)
      2. LV, RP, RV, until LV vac ≈ 1.0e-3 (Put new sample vials into the heater block)
      3. Inlet–Toggle C/O Valve, until 44 < 4e-11
      4. RM
      5. Check SP
      6. RI
      7. Manually operate 2 toggle valves (U: upper, L: lower) on reference gas cylinder
        1. L: pump out reference gas between the two valves 30s
        2. LV
        3. RP (Stage 2)
        4. L, U: release reference gas into the space between L and U 30s
        5. U, L: refill reference gas 30s (edit batch MPRPCM)
        6. L
      8. Measure reference gas
        1. RI, RP
        2. Set reference gas bellow to 0, until bellow fully open
        3. RM
        4. Check 44 ≈ 2.5e-9
        5. Test–Peak Center, close scan
    5. Edit batch MPRPCM
      1. Analysis–Autorun Batch Edit–select MPRPCM from the list
      2. MPRPCM: Setup
      3. Release all
      4. Enter new sample numbers, then OK–OK
    6. Reboot by Ctrl+Alt+Del
    7. Fill LN2 dewar
      • wear PPE
      • efficient transfer is achieved at 5–10 psi
      • filled to 90% of their net capacity to reduce the risk of spillage
      • 20L, record on the notebook
      • Check two tubes before putting them into the LN2 dewar: ~2 cm from the end of the wood stick
    8. Analysis–Autorun Start/Stop–select MPRPCM from the list, Run

    Friday Standby

    1. Check status: Finished Autorun
    2. Remove LN2 tubes from dewar
    3. Prep–Water Trap Chiller Off
    4. ML, AB (check: AC)
    5. LV, RP, RF, RV, RM, until LV vac ≈ 1.0e-3
      • optional: Set reference gas bellow to 3100 (Stage 2); RI, pump out, and then RI
    6. HV

    CO2 from organic matter

    1. Connect organic reference gas cylinder

      1. HV
      2. RP, RF, check RI
      3. Disconnect Carrara Z reference gas cylinder
      4. Connect organic reference gas cylinder

    2. Fill organic reference gas

      1. Set reference gas bellow to 1000
      2. Check SP
      3. LV, RP, RF, RV, RM until LV vac ≈ 1.0e-3
      4. Inlet–Toggle C/O Valve, until Mass-44 < 4e-11 A
      5. RM
      6. RI
      7. Manually operate 2 toggle valves (U: upper, L: lower) on organic reference gas cylinder
        1. L: pump out reference gas between two valves 30s
        2. L, U: release reference gas into the space between L and U 30s
        3. LV
        4. RP
        5. U, L: refill reference gas 30s
        6. L
    3. Check system status by measuring organic reference gas
      1. RI, RF (so that space between RF and SC can be used for sample gas later)
      2. RM
      3. Tune reference gas bellow (squeeze, larger value) ⟹ Mass-44 ≈ 6e-9
      4. Test–Peak Center, close scan
      5. RV
      6. Disable changeover ( Inlet-)
      7. Manually analyse against reference gas itself (DACP–CO2, Sample name: ORG GAS REF ZERO). Can do 4.
      8. !!! After run: Enable changeover

    4. Connect sample tube (during run for last sample/reference gas)

      1. Check RI
      2. Disconnect organic reference gas cylinder or last sample tube
      3. Cut glass sample tube, insert into metal tube (from the simple end)
      4. Connect (the more complicated end with a filter) to reference gas inlet
      5. LV, RI, RP, SP (space between RF and SF)

    5. Fill sample gas (after run for last sample/reference gas)

      1. Set sample gas bellow to 0
      2. LV, RI, RP, SP, SF, SV, SM (space between RF and SC) until LV vac ≈ 1.0e-3
      3. Inlet–Toggle C/O Valve, until Mass-44 < 4e-11 A
      4. LV
      5. Break sample tube, until Mass-44 is stable
    6. Measure sample gas
      1. RI, SF (store sample gas in between)
      2. Tune sample gas bellow (squeeze, larger value) ⟹ Mass-44 ≈ 6e-9, record the value
      3. Inlet–Toggle C/O Valve, tune reference gas bellow (squeeze, larger value) ⟹ Mass-44 sample − reference ≈ 0.08
      4. Inlet–Toggle C/O Valve, tune sample gas bellow (squeeze, larger value) ⟹ Mass-44 sample − reference ≈ 0.08
      5. RV, SV
      6. Manually analyse (DACP–CO2, Sample name: …)
      7. Can do 4 when the analysis for current sample gas appears to be OK
    7. Repeat 4–6
    8. Standby 1. (Check RI) Connect Carrara Z reference gas cylinder 2. Changeover → waste 3. Set both bellows to 0 4. Pump all that is left of SI: LV, RI…, until LV vac ≈ 1.0e-3 5. RI 6. HV

    Misc.

    • Cold fingers: click stop before clicking heat/cool
    • Scan config for OCR : 600 dpi, Gray scale, TIFF
    • Replenish reference gas:
      • Stage 1: RP, gradually decrease reference gas bellow (increase bellow size) through time
        • until 2024-01-23: 1400
      • Stage 2: RP, gradually decrease reference gas bellow (increase bellow size) through time
        • after 2024-01-23: 3100
    • Error 1051 is OK

    Troubleshoot

    Solenoid valve can not be turned on/off

    Symptoms:

    • Error: Too many small samples
    • Cannot pump out gas

    Solution:

    1. Turn off pumps, test valves
    2. Change solenoid

    Solenoid valve leak

    Symptom: hear the leak

    Solution: change (3) valve seals

    Data quality control

    • Ion Gauge for Sample and Reference are similar, difference < 0.2e-8
